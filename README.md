## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## To Edit an item

Click 2 times on a item

## To Delete an item

Open the "Edit Motal" and click on trahs icon

## To Add an item

Click on calendar (any empty space), then click on "Add" button inside "Create New Item Motal"
